</p>

## Features
- **100% Free**
- **High-Quality Music**
- **24/7 Uptime**
- **Voting System (/skip, /skipto, and /stop)**
- **Autoplay**
- **Player Buttons**
- **Custom Playlists**

## Supported Sources
- **YouTube**
- **Spotify**
- **Deezer**
- **SoundCloud**
- **Apple Music** 
- **Twitch** 
- **Bandcamp** 
- **Vimeo**
